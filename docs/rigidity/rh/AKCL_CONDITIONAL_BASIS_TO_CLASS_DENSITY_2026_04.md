# AKCL Conditional Basis-to-Class Density (2026-04)

## Status
Conditional.

## Standing model assumption
Use the coefficient-space model
\[
\mathcal A_{\mathrm{AKCL}}=\mathbb R^m.
\]

Define
\[
\mathcal P_{\mathrm{AKCL}}:=\mathbb R^m.
\]

## Exact density statement
For every
\[
a\in\mathcal A_{\mathrm{AKCL}}
\quad\text{and}\quad
\varepsilon>0,
\]
choose
\[
g:=a\in\mathcal P_{\mathrm{AKCL}}.
\]
Then
\[
\|a-g\|_{\mathcal A_{\mathrm{AKCL}}}=0<\varepsilon.
\]

Hence
\[
\overline{\mathcal P_{\mathrm{AKCL}}}^{\,\|\cdot\|_{\mathcal A_{\mathrm{AKCL}}}}
=
\mathcal A_{\mathrm{AKCL}}.
\]

## Finish condition
This density statement is exact only under the standing model assumption above.
